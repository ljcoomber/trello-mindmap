Exports Trello boards to Freemind Mind Maps.

To setup, type `make setup`. 

To run, make sure that `virtualenv` is active by typing `. env/bin/activate`, then `python trello_mindmap.py -o OUTFILE -b BOARD -k KEY -s SECRET`.
